# WreckIt! 魔性翻车 — 设计文档

**日期**: 2026-05-21
**类型**: 2.5D 侧视卷轴赛车游戏 · 无尽休闲模式
**栈**: Python 3.12 + pygame 2.6 · 零外部素材

## 一句话定位
轻松休闲的无尽地形物理赛车，玩家操控 1 辆怪车，在不断变化的生物群系（草原/冰原/月球/火山）里尽量开远，靠空中翻滚加油、看着自己魔性翻车找乐子。

## 核心玩法
- 侧视 2.5D，重力 + 摩擦 + 弹簧悬挂
- **油门 →** · **刹车/倒车 ←** · **车身倾斜 ↑↓** · **紧急刹车 空格** · **复活 R** · **暂停 ESC**
- 死法：燃料耗尽 / 司机被甩出 / 车体解体 / 碰到岩浆
- 死后慢动作回放最后 3 秒

## 5 大魔性元素
1. **司机布娃娃**：受到大冲击时连接断裂，被甩飞
2. **空中翻滚奖励**：每 360° 翻滚 +20% 油量 +500 分
3. **车身碎裂**：撞击超阈值时轮子/车顶飞出
4. **燃料倒计时**：开车持续耗油，路上捡油桶续命
5. **死亡回放**：慢镜头展示最后惨烈瞬间

## 3 辆车
| 车 | 加速 | 重量 | 油耗 |
|---|---|---|---|
| 小破皮卡 | 中 | 中 | 中 |
| 不靠谱赛车 | 高 | 轻 | 高 |
| 闷罐货车 | 低 | 重 | 低 |

## 4 生物群系（按距离每 500m 切换）
- **草原**：标准物理，草丘 + 小灌木
- **冰原**：摩擦 30%
- **月球**：重力 1/6，陨石坑地形
- **火山**：随机岩浆喷柱即死

## 难度恒定
难度不随距离上升，仅靠地形扭曲度提供天然挑战感——主打休闲。

## 物理实现要点（自写）
- 轮子：圆形质点，与地形线段做圆-段碰撞，沿法线弹出 + 沿切线施加摩擦
- 底盘：刚体（位置/速度/旋转/角速度），通过弹簧关节连接两轮形成悬挂
- 引擎：对后轮施加角加速度，靠摩擦推车前进
- 司机：圆形质点用距离约束挂在底盘上，约束在某阈值断裂
- 集成：半隐式欧拉，dt 固定 1/60s

## 验收标准
- `python main.py` 一键跑起
- 60 FPS 流畅，无明显抖动
- 4 个群系视觉差异明显
- 排行榜跨会话持久化
